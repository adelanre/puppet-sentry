Sentry Module
===

This module installs and configures the sentry application. 


What it does
---
Install; 
 * Sentry 8.4.0
 * redis-server 3.0.7
 * supervisor
 * python packages and 
 * nginx-full

```

This module assumes that a postgresql database exists and its login credentials have been added in Sentry Hiera file.
